# FundosRAG-CVM
O FundosRAG CVM é um sistema de AI Engineering que ingere todos os regulamentos, lâminas e prospectos de fundos de investimento publicados no Portal de Dados Abertos da CVM, processa esses documentos com uma arquitetura de processamento multimodal híbrida e disponibiliza as informações via chatbot RAG com avaliação automática de qualidade.
